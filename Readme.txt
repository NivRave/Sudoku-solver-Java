Sudoku solver using Backtracking algorithm
The program, programmed in JAVA, solves any valid 9x9 sudoku.
The input file is Sudoku.txt.
Original sudoku must be inside the 'Sudoku.txt' file, as a 9x9 matrix of numbers,
without spaces, commas or any seperators. An empty cell will be '0'.
An example puzzle is saved in the 'Sudoku.txt' file.
